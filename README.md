# Quiz

Небольшое desktop-приложение на Java Swing: викторина из 5 вопросов на русском языке.

## Что есть в приложении
- Система вопросов и ответов.
- Небольшое визуальное оформление (цвета, заголовок, индикатор прогресса).
- Короткое описание теста в верхней части окна.
- Показ результата и предложение пройти викторину заново.

## Запуск
```bash
javac QuizApp.java
java QuizApp
```
